Les balises DIV :

La balise div est une balise neutre de type block. Elle me servira pour me structure ma page

Les balises de type block agissent sur un axe vertical et donc génère un retour à la ligne.
Je pourrais toutefois changer leur comportement en CSS.

Les balises de type Inline agissent sur un axe horizontal et donc ne génère aucun retour à la ligne.
Je pourrais toutefois changer leur comportement en CSS.

Les formulaires :

  • La balise FORM permet de créer un formulaire
  • name : permet de donner un nom au formulaire
  • action : attribut obligatoire ! désigne la page ou se situe le script qui va recevoir et traiter les données
  • method :permet de préciser quelle méthode [GET | POST ] doit être utilisé par le navigateur pour transmettre les données saisies vers la page "action".
